Ÿ   官网下载最新的SuperMap iClient3D 9D(2019) for WebGL产品包

打开SuperMap官网地址http://support.supermap.com.cn/：

点击“软件下载”，打开软件下载界面：

点选“网络客户端”→“SuperMap iClient3D 9D(2019)” →“(V9.1.0)”→“SuperMap iClient3D 9D(2019) for WebGL”，打开SuperMap iClient3D 9D(2019) for WebGL产品下载界面：

 

将鼠标移至“离线安装包”，选择“官网下载”或“百度云盘下载”，将SuperMap iClient3D 9D(2019) for WebGL产品包下载至本地。

Ÿ   官网下载最新的iServer产品包，部署、启动

参照上一步骤，在官网打开软件下载界面，

点选“云GIS”→“SuperMap iServer 9D(2019)”→“(V9.1.0)”→“SuperMap iServer 9D(2019) for Windows(64位)(Deploy包)”，打开SuperMap iServer 9D(2019) for Windows(64位)(Deploy包)产品下载界面：

将鼠标移至“离线安装包”，选择“官网下载”或“百度云盘下载”，将SuperMap iServer 9D(2019) for Windows(64位)(Deploy包)产品包下载至本地。

将SuperMap iServer 9D(2019) for Windows(64位)(Deploy包)产品包解压放到指定路径。本说明将解压后的产品包放到E:\soft\路径下：

打开“bin”文件夹，双击startup.bat启动iServer:

iServer成功启动界面：

Ÿ   SuperMap iClient3D 9D(2019) for WebGL产品部署

将 SuperMap iClient3D 9D(2019) for WebGL产品包解压放到路径E:\soft\supermap-iserver-9.1.0-win64-deploy\webapps路径下。

打开examples文件下，文件夹包括示范程序的html文件，可以查看范例代码。
